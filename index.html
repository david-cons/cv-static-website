> projects

Indoor Navigation System | University project

This project aimed to enhance warehouse efficiency by developing an indoor navigation system for warehouse pickers. The system provides real-time worker localization, optimizing the picking process. The system is made up of 3 components. The mobile application, built using React Native, allows workers to view optimized routes to their assigned locations within the warehouse. The backend, powered by Laravel, processes internal API requests for new orders, order assignment, and general path optimization using a logical model of the warehouse. And a Python module used for indoor localization using a Bluetooth beacon system

The project was developed in collaboration with Scrumble and PolyesterShoppen BV, incorporating feedback from industry users to ensure practical application. This was within the context of 2IPE0 | Final Software Engineering Project course at the TU/e

Tech Stack: React Native, Laravel, MySQL, PHP, Python, Docker
